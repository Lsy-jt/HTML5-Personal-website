用H5 canvas标签画一只 皮卡丘！！
编辑时间：2019/6/09 17:30:22 作者：LLLsy 浏览量：12

小赌为快




    前言：如果你没使用过canvas标签或是HTML5，那么请简略欣赏皮卡丘！

    因为最近也上映 <大侦探皮卡丘>，所以也想用代码画一只皮卡丘试试看。


    如果你有感兴趣的话，就请接着往下看吧！！


    起初想着自己用，就简简单单的画个火柴人，后来想着自己还没用canvas画过自己想要画的东西，不如把这个标签用起来！


   分析：

canvas做圆，可画眼珠子，腮红 。
做扇形，完成鼻子。
做半椭圆，完成嘴巴。


    效果图如下： 

var c=document.getElementById("myCanvas");

var cxt=c.getContext('2d');        /* 声明cxt变量，带.getContext("2d")动作 */
 
//右眼

cxt.fillStyle="#000";          /* .fillStyle 填充属性 */

cxt.beginPath();                /* .beginPath() 开始路径 */

cxt.arc(200,188,30,0,Math.PI*2,true);  /* .arc()  绘制圆，参数较多，下面有详解 */

cxt.closePath();                /* .closePath()  闭合路径 */

cxt.fill();                    /* .fill();执行填充 */      


cxt.fillStyle="#FFF";

cxt.beginPath();

cxt.arc(188,175,11,0,Math.PI*2,true);

cxt.closePath();

cxt.fill();

     这一段代码首先获取画布，填充背景颜色为黄色，开始绘图，以（200,180）坐标为圆心，从0度角开始，顺时针整一个半径为30像素的圆。


    接下来类似： 

cxt.arc(450,188,30,0,Math.PI*2,true);//左眼
cxt.arc(438,175,11,0,Math.PI*2,true);

cxt.arc(488,275,35,0,Math.PI*2,true);//两个晒红
cxt.arc(162,275,35,0,Math.PI*2,true);

cxt.moveTo(328,200);         //鼻子
cxt.arc(328,200,11,Math.PI+0.5,Math.PI*2-0.5,false); 

cxt.scale(1,2);//嘴巴
cxt.arc(328,220/2,50,0,Math.PI,0); 
cxt.arc(300,220/0.75,27,0,Math.PI,0);
cxt.arc(356,220/0.75,27,0,Math.PI,0);
    参数说明：

    arc()： 变量依次是圆心的X坐标，Y坐标，半径长度，起始角度，结束角度，顺/逆时针。
